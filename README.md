# Baseline

Grid Generator for designers and developers.

**Live:** https://samhayek-code.github.io/baseline

## v3.0 Updates

- Collapsible grid categories
- Spacing sliders with number inputs (margin, padding, gutter)
- Filled slider tracks
- Animated canvas resizing
- Custom dropdown for canvas size presets
- Randomize with weighted distribution (rare 4th grid)
- 2-layer system with independent settings per layer
- Layer opacity control
- Mobile responsive layout (bottom drawer)
- Export button with shimmer animation
- Smoother zoom (3% increments)
